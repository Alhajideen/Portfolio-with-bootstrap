# My Portfolio

A simple portfolio website developed using bootstrap 5.2. It has responsive capabilities and consists of the following sections:

- Navbar
- Headline
- Projects
- About
- Contact

The layout of the elements has been made using the Bootstrap framework.


## Built With

- HTML5
- Bootstrap
